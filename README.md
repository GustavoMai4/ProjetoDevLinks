# ProjetoDevLinks

projeto criado para aprender os fundamentos de HTML, CSS e JS com a Rockeatseat.

o projeto contem como exemplos os links das redes sociais do professor Clovis de Barros Filho e alguns links da revista Inspire-c, do canal Sócios e da Casa do Saber.

Também possui duas imagens do Clovis.

Esse projeto foi desenvolvido com as seguintes tecnologias:

HTML e CSS
JavaScript
Git e Github
Figma
